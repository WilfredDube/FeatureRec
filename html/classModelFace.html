static void ModelFace::classifyFaces ( std::vector< ModelFace > &  faces)
inlinestatic

Identifies FACE and THICKNESS_DEFINING_FACE faces types. Retrieves the BEND_FACE type faces and compares its STRAIGHT_LINE edge type edge to all the PLANAR type face. If there is a matching edge then the bend is connected to that face. The joining face ID is set to the connected face's ID. The connected face's FaceType is also set to FACE.
